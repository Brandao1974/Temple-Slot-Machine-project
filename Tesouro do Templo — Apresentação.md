# 🎰 Tesouro do Templo — Apresentação

## 🌍 Visão

Tesouro do Templo é um jogo estilo slot machine com temática de exploração mística em ruínas antigas, projetado para funcionar como aplicação web instalável (PWA).

---

## 🎮 Diferenciais

* Funciona como app (instalável)
* Compatível com mobile, tablet e desktop
* Arquitetura escalável
* Design modular (fácil expansão)

---

## 🧠 Tecnologia

* HTML, CSS, JavaScript
* PWA (Progressive Web App)
* Engine própria de slot
* Sistema visual em camadas

---
